# Typescript Helpers v0.1.1

Provides helper functions and extension methods for Typescript

## Installation:

In npm:

```shell
$ npm i @evan-abc/typescript-helpers --save-dev
```

In yarn:

```shell
$ yarn add -D @evan-abc/typescript-helpers
```

## Extension Methods

### Array

First:

```typescript
const arr = [1,2,3]
arr.first() // 1
```

Last:

```typescript
const arr = [1,2,3]
arr.last() // 3
```
